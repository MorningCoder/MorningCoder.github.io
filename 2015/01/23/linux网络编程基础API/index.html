依稀记得大一时对网络编程强烈的好奇心，非常想知道如何将信息从本机传递到另一台机器。现在上了两年半大学，学了一学期计算机网络，可算是在网络方面有了一点理解。这个假期准备开始好好研究linux下的网络编程，顺便写写学习笔记。

linux下网络编程的核心也是基础就是他提供的一堆系统调用，说白了就是一堆函数，学习这些基础API的用法对日后真正构建服务器有很大帮助。
