    • 函数

    函数这个概念在大多数编程语言中都广泛存在。它既可以作为过程式语言的基本单位，同时也可以扩展到类中，作为类方法，成为面向对象体系中的一个重要部分。python是一个完全面向对象的语言，但是也支持定义普通函数。这种使用方法同C\C++中大抵相同，都是作为一段可重用的代码段出现的。它的相关语法也很简单，只要熟悉一门编程语言，看一下即可掌握。

    一些高级用法

    python中变量的作用域和C中是一样的，在函数中定义的同名变量会覆盖外部变量，对其的修改不会影响外部变量。但是也可以直接在函数内修改一个外部变量的值，但这样会影响代码的可读性，其他人可能会分不清变量到底是函数内的还是全局的。因此python提供了一个关键字是：global。它的作用是显示指明某一个变量是在外部定义的，也就是指明它是全局的（global）。

    模块

    所谓python的模块，我觉得就类似于C++的命名空间，是一个函数、变量以及类定义的集合。在python模板中，它也定义了一些内置的模块，提供了很丰富的功能供我们使用。至于标准库我们后面再说。模块的文件当然要一.py结束，其实就是一个python代码文件。我们看下如何使用标准库定义的sys（system）模块：

    类似java，引入模块使用import语句。当引入模块sys时，解释器会自动在sys.path制定的目录中自动寻找sys.py模块，找到后初始化即可使用。也可以看到python同样使用句点（.）来引用模块内变量。至于这个sys模块后面我们会说到，现在只用知道它可以用来获取传递给本程序的命令行参数，就是类似于C++的main函数的参数。还有一个语句是“from…import”，这个用处类似于C++的using语句，例如：from sys import argv，意思跟using一样。
