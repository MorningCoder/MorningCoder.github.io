Python学习笔记（1）：Pyhon简介

2014-09-29

很早之前就听说过Python这个脚本语言很强大，作为一个程序猿还是必须要学习的，最近就开始研究一下，看的一些东西整理在这里了。

Python是一个简单易学但是功能强大的脚本语言，属于解释性语言，并且是开源的，它的最大目的就是让使用者能够专注于使用语言去解决问题，而不是搞明白语言本身。python也是一个高层语言，也就是不必考虑底层内存分配的细节，这对C++的使用者来说的确方便不少。由于python的开源性质，使得跨平台使用它很简洁，Python现在已经被移植在很多平台上，包括Windows、Linux、FreeBSD、Mac等等。

python同时支持面向过程以及面向对象编程方式，在面向对象方面python与C++和Java相比更加简单和强大。同时Python具有丰富的类库，包括Web、GUI、数据库、多线程操作等等，这给使用者带来了很多便利。

python还有一个重要特性就是可嵌入性和扩展性，可以把Python嵌入C/C++程序中并且可以运行良好，所以python也被成为”胶水语言”。
